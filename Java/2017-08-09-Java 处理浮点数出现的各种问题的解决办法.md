**一、**SpringMvc使用注解PathVariable获取有句号或小数点的参数时，取值不全

使用：


```java
@RequestMapping(value = "{price:.+}",method = RequestMethod.GET)
```





**二、**SQL语句求平均值保留2位小数


```java
cast(AVG(price) AS DECIMAL(15,2))
```
